Название метода

Аналитический

Суть метода на языке ученика

Используются логические рассуждения; составляются выражения с буквами; применяются формулы (и даже уравнения)

Возьми на заметку

Решение задачи начинай со слов «Что нужно знать, чтобы ответить на вопрос данной задачи (выполнить ее требование)?»
